Web scrapping Overwatch players stats data with BeautifulSoup

Overwatch is a First Person Shooter (FPS) videogame developed by Blizzard and released on May 24, 2016. With an estimated of 8.7 million active players. Using the module BeautiFulSoup we will scrape data related to the Top 100 Overwatch players on PC based on skill rating system from https://overwatch.op.gg/leaderboards. Note: After the realease of Overwatch 2 the aforementioned website is no longer able to display player stats. The code below it will not work if you use it to retrieve data from the website.

First we import the required modules for web scrapping

import pandas as pd
import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt
import requests
from bs4 import BeautifulSoup

We start web scraping the players data from Overwatch OP website using the module get from requests

url = "https://overwatch.op.gg/leaderboards"
data = requests.get(url, headers = {"User-agent":"Super Bot Power Level Over 9000"}).text 
#Some websites do not allow you to scrape unless you declare that you are a bot#
#It is also important to note that if there is no API on a website it means "Don't scrape me!!"  
#Be considerate while web scraping and do not saturate other people's servers ;-)#

We use the BeautifulSoup module to pull the data from the html code of the website

soup = BeautifulSoup(data, "html5lib")
#This gives us a BeautifulSoup object, which represents the data as a nested data structure
html_data = pd.read_html(str(soup))
print(html_data[0:2])
